# LuxeMart

Full-stack Luxury E-Commerce application with MongoDB, Express, and React.

## Setup

1. **Backend**:
   - `cd backend`
   - `npm install`
   - `npm run dev`

2. **Frontend**:
   - `cd frontend`
   - `npm install`
   - `npm run dev`

3. **Seeding**:
   - Visit `http://localhost:5000/api/seed` in your browser to pre-populate products.
